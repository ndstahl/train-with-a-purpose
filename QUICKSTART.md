## Deploy to Vercel (3 Easy Steps)

### Step 1: Push to GitHub

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit your code
git commit -m "Initial commit - Train With A Purpose website"

# Create a new repository on GitHub.com
# Then add it as remote and push:
git remote add origin https://github.com/YOUR_USERNAME/train-with-a-purpose.git
git branch -M main
git push -u origin main
```

### Step 2: Connect to Vercel

1. Go to https://vercel.com
2. Click "Sign Up" and use your GitHub account
3. Click "New Project"
4. Select your "train-with-a-purpose" repository
5. Click "Deploy"

### Step 3: Done!

Your site will be live in about 2 minutes at a Vercel URL like:
`https://train-with-a-purpose.vercel.app`
